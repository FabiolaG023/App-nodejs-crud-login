# App-nodejs-crud-login
Aplicacion creada a partir de Nodejs, se trata de un CRUD hecho en JavaScript, con autentificacion; haciendo uso del motor de vistas Pug y del ORM Sequilize
Dependencias
npm install 
   express
   express-sessions
   Body-Parser
   bcrypt
   Sequilize
   pug
   mysql2
   method-override
